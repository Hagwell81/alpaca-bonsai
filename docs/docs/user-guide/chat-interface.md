# Chat Interface

Using the Alpaca chat UI.

## Overview

The chat interface provides a conversational AI experience with:
- Streaming responses
- Code highlighting
- Message history
- Markdown rendering

## Sending Messages

Type in the input box and press Enter (or Shift+Enter for new line).

## Streaming

Responses appear in real-time as the model generates tokens.

## Code Blocks

Code appears with syntax highlighting and copy button:

```python
print("Hello World")
```

## Message Actions

- **Copy**: Copy message text
- **Regenerate**: Regenerate response
- **Delete**: Remove message
- **Edit**: Modify your message

## Conversation Management

- New conversation: Ctrl+N
- Rename conversation: Click title
- Delete conversation: Right-click in sidebar
