# Model Management

Manage AI models in Alpaca.

## Downloading Models

Use the built-in model browser:
1. Open Models panel
2. Browse or search
3. Click Download
4. Wait for completion

## Importing Models

Import custom GGUF files:
1. File > Import Model
2. Select .gguf file
3. Model appears in list

## Selecting Models

Click a model in the sidebar to load it.

## Model Info

View details:
- Parameters count
- Context length
- File size
- Quantization level

## Deleting Models

Right-click a model and select Delete to free disk space.
