# Application Menu

Overview of the Alpaca application menu.

## Menu Structure

### File Menu
- **New Conversation** — Start a new chat
- **Open Model Folder** — Browse model files
- **Import Model** — Import a GGUF model
- **Export Chat** — Save conversation to file
- **Preferences** — Open settings
- **Quit** — Exit application

### Edit Menu
- **Undo/Redo** — Standard text editing
- **Cut/Copy/Paste** — Clipboard operations
- **Select All** — Select all text
- **Find** — Search in chat

### View Menu
- **Reload** — Refresh UI
- **Toggle Full Screen** — Full screen mode
- **Zoom In/Out** — Adjust UI scale
- **Reset Zoom** — Default zoom level

### Conversation Menu
- **New Chat** — Start fresh conversation
- **Clear History** — Remove all messages
- **Save Conversation** — Export to file
- **Load Conversation** — Import from file

### Help Menu
- **Documentation** — Open built-in docs viewer
- **Keyboard Shortcuts** — Show shortcut reference
- **Check for Updates** — Manual update check
- **About** — Application information

## Customization

Menu items can be customized through:
- Settings > Interface > Menu Options
- Custom keyboard shortcuts
- Plugin extensions

## Platform Differences

### macOS
- App menu contains About and Preferences
- Standard macOS edit menu
- Window menu for managing windows

### Windows/Linux
- File menu contains Preferences
- Standard edit and view menus
- Help menu with documentation
