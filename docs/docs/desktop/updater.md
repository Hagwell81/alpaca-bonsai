# Auto Updater

Keep Alpaca up to date automatically.

## Overview

The auto updater checks for new versions and installs them seamlessly.

## Update Sources

- **GitHub Releases**: Primary source for updates
- **Manual Check**: Check for updates on demand

## Configuration

### Automatic Checks

Settings > Updates:
- Check for updates automatically
- Update frequency: Daily/Weekly/Monthly
- Download updates automatically
- Install on quit

### Channel Selection

- **Stable**: Production releases
- **Beta**: Pre-release versions
- **Dev**: Development builds

## Manual Update Check

1. Go to Help > Check for Updates
2. Or use tray menu
3. View update details
4. Click Install to update

## Update Process

1. Check for new version
2. Download update
3. Verify signature
4. Install on next launch
5. Restart application

## Troubleshooting

### Update Check Failed

- Check internet connection
- Verify update channel
- Check proxy settings
- Review firewall rules

### Download Failed

- Check disk space
- Verify network stability
- Try manual download

### Installation Failed

- Run as administrator
- Close all app instances
- Check antivirus settings
- Manual install from releases

## Disabling Updates

Settings > Updates > Disable automatic updates

**Note**: Manual updates are still available.

## Release Notes

View what's new:
- Help > Release Notes
- Shows version history
- Lists new features
- Documents bug fixes
