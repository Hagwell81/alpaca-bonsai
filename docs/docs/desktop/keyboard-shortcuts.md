# Keyboard Shortcuts

Quick reference for Alpaca keyboard shortcuts.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + N` | New conversation |
| `Ctrl + ,` | Open settings |
| `Ctrl + Q` | Quit application |
| `Ctrl + Shift + I` | Toggle developer tools |
| `F11` | Toggle full screen |

## Chat Shortcuts

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift + Enter` | New line |
| `Ctrl + /` | Focus input |
| `Ctrl + Up` | Previous message |
| `Ctrl + Down` | Next message |

## Navigation Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + 1-9` | Switch conversation |
| `Ctrl + Tab` | Next tab |
| `Ctrl + Shift + Tab` | Previous tab |
| `Ctrl + W` | Close current tab |

## Developer Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + Shift + C` | Inspect element |
| `F5` | Reload window |
| `Ctrl + R` | Reload window |
| `Ctrl + Shift + R` | Force reload |

## macOS Equivalents

| macOS | Windows/Linux |
|-------|---------------|
| `Cmd` | `Ctrl` |
| `Cmd + ,` | `Ctrl + ,` |
| `Cmd + Q` | `Ctrl + Q` |
| `Cmd + W` | `Ctrl + W` |

## Customization

Keyboard shortcuts can be customized in Settings > Keyboard:

1. Open Settings (Ctrl+,)
2. Navigate to Keyboard tab
3. Click on action to modify
4. Press desired key combination
5. Click Save

## Disabling Shortcuts

To disable a shortcut:

1. Open Settings > Keyboard
2. Find the shortcut
3. Click the X icon
4. Save changes
