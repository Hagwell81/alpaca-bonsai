# Quick Start

Get up and running with Alpaca.

## First Launch

1. Install the application
2. Launch Alpaca
3. Wait for the backend to initialize
4. Select or download a model
5. Start chatting

## Downloading Models

Built-in model manager:
- Open the Models panel
- Browse available models
- Click Download on desired model
- Wait for download to complete

## Your First Chat

1. Select a loaded model
2. Type a message in the input box
3. Press Enter to send
4. Watch the AI respond
