# Project Awareness

Alpaca provides project-level context to IDEs.

## Features

- File tree awareness
- Symbol indexing
- Import resolution
- Code context injection

## How It Works

1. IDE opens workspace
2. Alpaca indexes files
3. Context sent with each request
4. AI understands project structure

## Configuration

Enable in IDE settings:
- File indexing
- Symbol resolution
- Context window management

## Benefits

- Better code suggestions
- Accurate import handling
- Relevant documentation
- Project-specific answers
