# IDE Keyboard Shortcuts

Quick reference for IDE integration shortcuts.

## Cursor Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + L` | Open chat |
| `Ctrl + K` | Inline edit |
| `Ctrl + I` | Command palette |
| `Tab` | Accept suggestion |

## VS Code Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + Shift + P` | Command palette |
| `Ctrl + I` | Inline chat |
| `Ctrl + Shift + L` | Chat sidebar |
| `Ctrl + Enter` | Submit request |

## Custom Shortcuts

Configure in IDE settings for your preferred bindings.
