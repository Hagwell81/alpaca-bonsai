# Welcome to Alpaca

**Alpaca** is a comprehensive, open-source local AI platform that brings together chat, development tools, and agentic services in a unified desktop application.

## What is Alpaca?

Alpaca is designed for developers and AI enthusiasts who want to run AI models locally while maintaining the flexibility to connect to cloud providers. It provides a modern, intuitive interface for interacting with large language models (LLMs) without relying on external APIs or sending your data to third-party services.

## Key Features

- **Local Model Execution** — Run models directly on your hardware with llama.cpp for maximum privacy and zero latency
- **Multi-Provider Support** — Seamlessly switch between local inference and cloud providers (OpenAI, Google, Anthropic, Mistral, and more)
- **Agentic Services** — Create and orchestrate AI agents with tool use, skill composition, and subagent delegation
- **Knowledge Base & RAG** — Build knowledge bases from web content, documents, and images with vector search
- **Integrated IDE** — Built-in code editor, terminal, and Copilot-like assistance for development workflows
- **Desktop-First Design** — Native desktop application with system tray integration, offline documentation, and keyboard shortcuts
- **Cross-Platform** — Available on Windows, macOS, and Linux with automatic hardware detection (CUDA, ROCm, Vulkan, CPU)
